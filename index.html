💡 Negative Indexes: Count backwards from the end!
list[-1] = LAST item  |  list[-2] = second to last
inventory = ["sword", "potion", "shield", "key"] print(inventory[0]) # → "sword" (1st item) print(inventory[2]) # → "shield" (3rd item) print(inventory[-1]) # → "key" (LAST item!) print(inventory[-2]) # → "shield" (2nd from end)

⚡ Chapter 3 · List Superpowers (Methods!)
⚡
Superpowers for Your List!
append, remove, pop, len, sort
Theory
🧙 SAGE CODY:Lists have built-in SUPERPOWERS called methods! You use them with a dot: mylist.method(). They let you add items, remove items, sort them, and more!
inventory = ["sword", "potion", "shield"] # ➕ ADD to the end: inventory.append("bow") # → ["sword", "potion", "shield", "bow"] # ➕ INSERT at a position: inventory.insert(1, "key") # → ["sword", "key", "potion", "shield", "bow"] # ❌ REMOVE by name: inventory.remove("potion") # 🗑️ POP removes last (or by index): last_item = inventory.pop() # 📏 LEN counts items: print(len(inventory)) # → 3 # 🔡 SORT alphabetically: inventory.sort() # 🔄 REVERSE the order: inventory.reverse()
🎮 Quick Reference:
.append(x) → Add x at end  |  .insert(i, x) → Add x at position i
.remove(x) → Delete x  |  .pop() → Remove & return last item
len(list) → Count items  |  .sort() → Sort it!

✂️ Chapter 5 · Slicing — Cut the List!
✂️
Grab a Piece of the List!
list[start : end]
Theory
🧙 SAGE CODY:Sometimes you only want PART of a list. Slicing lets you cut out a section! It's like taking only the first 3 items from your inventory instead of all of them.
heroes = ["Aria", "Zara", "Kira", "Nova", "Luna"] # [0] [1] [2] [3] [4] # First 3 items (index 0, 1, 2): print(heroes[0:3]) # → ['Aria', 'Zara', 'Kira'] # From index 2 to end: print(heroes[2:]) # → ['Kira', 'Nova', 'Luna'] # From start to index 3: print(heroes[:3]) # → ['Aria', 'Zara', 'Kira'] # Last 2 items: print(heroes[-2:]) # → ['Nova', 'Luna']
🔑 Remember: list[start:end] — the END index is NOT included! Like [0:3] gives items 0, 1, 2 (NOT 3)!

# Normal way (4 lines): numbers = [1, 2, 3, 4] doubled = [] for n in numbers: doubled.append(n * 2) # COMPREHENSION way (1 line!) 🪄: doubled = [n * 2 for n in numbers] # → [2, 4, 6, 8] # With a condition — only even numbers: evens = [n for n in numbers if n % 2 == 0] # → [2, 4]
